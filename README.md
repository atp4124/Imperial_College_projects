# Imperial_College_projects

This repo contains projects that were completed during the BSc Mathematics at Imperial College London.

- Under *Fundamentals of Data Science*, we investigate Random Forest Classifier, SVM, Graph analysis, CNNs and MLP.
- Under *Scientific Computation*, we investigate algorithms like breadth first search, dijkstra, depth search, random walk simulations on graphs, solution to a linear diffusion problem, PDEs system of equations for epidemics, PCA using SVD, compact finite difference scheme
